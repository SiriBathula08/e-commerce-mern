# ShopElite — Full Stack E-Commerce Platform

## Overview

ShopElite is a production-ready MERN stack e-commerce application supporting product browsing, cart management, and secure online payments.

## Features

* JWT authentication
* Admin dashboard
* Product management
* Order tracking
* Secure online payments

## Tech Stack

Frontend

* React.js
* Material UI
* Bootstrap

Backend

* Node.js
* Express.js

Database

* MongoDB Atlas

Payments

* Razorpay

Deployment

* Vercel (Frontend)
* Render (Backend)

## API Features

* User authentication APIs
* Product management APIs
* Cart management APIs
* Order and payment APIs

## Installation

Backend

```bash
npm install
npm start
```

Frontend

```bash
npm install
npm start
```
